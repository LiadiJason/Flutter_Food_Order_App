# Online Food Ordering System
A food ordering mobile application system that allows users to order food with several features.<br /> <br />
***List of features:***
- Login and Logout
- Filtered categories
- Swipe gestures
- Payment<br /><br />

## Technologies Used <br />
- Dart
- Firebase
- Flutter
- Razorpay
